jQuery slider plugin - Eclipse

1. 개요

이미지 및 컨텐츠 롤링 기능이 있는 jquery 슬라이더 플러그인 입니다.
오늘도 영문사이트 보시느라 눈이 아플 한국의 웹퍼블리셔 및 웹개발자들을 위해 만들어 봤습니다.
사용자피드백을 통해 지속적으로 업데이트 할 예정입니다.

소스 다운로드 (github)

2. 옵션표

옵션명 옵션값 타입 옵션 설명 기본값
startIndex number 슬라이더 시작 인덱스 값 지정 0
arrow boolean 화살표 네비게이션의 사용 여부 true
pager boolean 단추 네비게이션의 사용 여부 true
slidesToShow number 한번에 보여질 슬라이드의 갯수 1
slidesToMove number 한번에 이동될 슬라이드의 갯수 1
speed number 슬라이드 전환 속도
(단위 : ms)		 500
margin number 슬라이드간 간격
(단위 : px)		 0
friction number 마우스드래그 및 터치무브로 슬라이드 전환시 감도 조절
(단위 : px)		 200
countIndex number 첫번째 활성 슬라이드의 인덱스값을 조절 0
autoplay boolean 오토플레이 사용 여부 false
interval number 오토플레이 전환 간격
(단위 : ms)		 3000
